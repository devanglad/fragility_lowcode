# fragility_lowcode

**Data is fitted on:**

(a) 3, 6, and 9-storey welded steel moment resisting frame with WUF-B connection [Gupta 1999]

(b) 3, 6, and 9-storey welded steel moment resisting frame with flush end plate and extended plate connection (bolted frame) [Gupta 1999]

(c) 2, 3, 4 and 6-storey reinforced concrete frame [Bracci et al 1997]

# -----------------------------------------------------

Regression coefficients for fitted logistic regression for governing local EDP and global EDP for all building typology. These include low-rise steel moment resisting frames (MRF) and low-ductile Reinforced concrete frames

$P = \frac{e^{\beta_0 + \beta_1 \log(IM) + \beta_2 \eta + \beta_3 N_s}}{1 + e^{\beta_0 + \beta_1 \log(IM) + \beta_2 \eta + \beta_3 N_s}}$
